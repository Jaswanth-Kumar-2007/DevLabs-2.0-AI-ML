# Week 1 — Build Your First AI Agent 🤖

## Task
Create a domain-specific AI agent (e.g. Sales Agent, HR Agent, Support Agent) with access to exactly 3 tools.

## Requirements
- Agent must be domain-specific (pick any domain — Sales, HR, Legal, Finance, etc.)
- Must have exactly 3 tools (e.g. search, calculator, email sender, CRM lookup, etc.)
- Agent should take a user goal as input and autonomously decide which tools to use

## Example
A Sales Agent with these 3 tools:
- get_lead_info(name) — fetches lead details
- send_email(to, subject, body) — sends a follow-up email
- calculate_discount(price, percent) — calculates deal pricing

## Submission Structure
Create a folder with your name inside Week-1/ and add your files there.

## Tips
- Keep the tools simple, mock/dummy implementations are totally fine
- Focus on the agent loop — how it reasons and decides which tool to call
- Comment your code so your thought process is clear
